Date

add(Number[, interval])
(반환형식 Date)
"2013-05-05".date().add(3); -> "Sat May 08 2013 00:00:00 GMT+0900"
"2013-05-05".date().add(2, 'm'); -> "Fri Jul 05 2013 00:00:00 GMT+0900"
"2013-05-05".date().add(2, 'y'); -> "Tue May 05 2015 00:00:00 GMT+0900"
diff(Date), diff(String)
(반환형식 Number)
"2013-05-05".date().diff("2013-05-08"); -> 3
"2013-05-05".date().diff("2013-05-08".date()); -> 3
print([format])
yyyy:년도, mm:월, dd:일, hh:시, mi:분, ss:초, dw:요일 을 조합하여 format으로 지정하면 그에 맞는 날짜형식 문자열이 반환됩니다. (반환형식 String)
"2013-05-05".date().print(); -> "2013-05-05"
"2013-05-05".date().print('yyyy년 mm월 dd일'); -> "2013년 05월 05일"
"2013-05-05".date().print('yyyy년 mm월 dd일 (dw)'); -> "2013년 05월 05일 (일)"
