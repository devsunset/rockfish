AXISJ 제이쿼리(JQuery) 라이브러리를 기반으로 하는 웹 애플리케이션을 제작을 위한 자바스크립트 라이브러리입니다. (Javascript UI Library based on JQuery) 웹 애플리케이션 이나 웹사이트 제작 시 사용되는 HTML, CSS 및 Javascript 기술을 이용하여 웹 브라우저(Web Browser) 또는 HTML를 이용하여 구동되는 다양한 디바이스에서 동작합니다. 액시스제이를 이용하여 일반적으로 웹에서 구현하기 어려운 다양한 기능과 UI 디자인을 제공하여 보다 쉽고 빠르게 웹 애플리케이션을 제작할 수 있습니다. "액시스제이는 개발자를 위한 코드 입니다." AXISJ 는 axis(축) J(javascript) 의 뜻이지만 애플리케이션 개발경험 Application eXperience AX 를 구현한다는 의미도 함께 담고 있습니다. 액시스제이는 자바스크립트 클래스 코드를 압축하여 하나의 파일로 처리 하지 않고 개발자가 필요한 클래스를 선택적으로 사용할 수 있도록 소스와 파일을 제공하고 소스코드도 압축하지 않고 개발자가 소스코드를 이해하고 개발할 수 있도록 하는데 초점을 맞추고 있습니다. 압축되지 않은 소스코드를 이해 하면서 궁극적으로 액시스제이를 사용하는 개발자들의 개발레벨이 모두 함께 성장해 나가기를 희망합니다. "액시스제이는 대한민국 개발자를 위한 코드 입니다." 대한민국 개발자가 대한민국에서 만든 대한민국 자바스크립트 UI 라이브러리 입니다. UI 라이브러리 사용 중 생기는 궁금증과 개선사항을 친구에게 물어보듯 물어볼 수 있습니다. 이해가 안되면 될 때까지 설명도 해드림니다. 무엇보다 이러한 개발자들의 아이디어와 요구사항을 기반으로 하여 자바스크립트 UI 라이브러리가 확장되어 나아간다는 것 입니다. 현재 버전이 해외 유명 자바스크립트 UI 라이브러리에 비해 화려하지 않을 수 있겠지만 대한민국 표준이 세계표준이 될 그날까지 매일 매일 묵묵히 한걸음을 내딛겠습니다. "액시스제이는 무료 입니다" 액시스제이를 이용하여 웹애플리케이션 이나 웹사이트를 제작하는데 아무런 제약이 없습니다. 저작권 표시 조차 필요 없습니다. 다만 액시스제이 소스코드를 임의로 수정하여 재 배포 하는 일은 허용되지 않습니다. (액시스제이를 통해 액시스제이의 이름으로 제공되는 경우는 무관합니다.) 명칭 액시스제이(AXISJ)는 "자바스크립트의 축(Axis of Javascript)"라는 의미와 Application eXperience AX를 지향한다는 의미를 담고 있습니다. 액시스제이는 다음과 같은 명칭으로도 사용됩니다. 액시스제이 AXISJ AX:J AXJ 액시스제이 클래스 파일은 "AX"라는 접두어 또는 네임스페이스(namespace)를 사용하고 있습니다. 요구사항 액시스제이(AXISJ)는 웹 브라우저(Web Browser)에서 동작하기 때문에 다음의 웹 브라우저 실행환경이 필요합니다.
