AXSlideViewer는 deskTop, mobile을 모두 지원하는 이미지 뷰어 입니다. 터치 혹은 마우스로 이미지를 이동 및 확대 축소 시킬 수 있습니다.

  • 이미지를 더블클릭하면 이미지가 확대/축소 됩니다.
  • 이미지를 클릭하면 컨트롤러 활성 비활성 처리 됩니다.
  • 마우스 휠을 조작하면 이미지가 확대/축소 됩니다.
  • 끝으로 이미지뷰어 하단 중앙에 핸들은 한번 눌러 보시면 압니다.
이미지뷰어에 기능이 맘에 안들거나 보강을 원하시면 깃 허브 이슈에 남겨주세요. 감사합니다.
