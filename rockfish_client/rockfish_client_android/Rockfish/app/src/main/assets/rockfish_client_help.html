ROCKFISH_LOGIN - 로그인

General Send 버튼 클릭 하여 테스트

ROCKFISH_LOGOUT- 로그아웃

General Send 버튼 클릭 하여 테스트

---------------------------------

IDX,TEMP1,TEMP2,TEMP3 4개의 필드 존재

테이블에 서비스에 따른 처리 수행

IDX는 Auto Increment Type 정수 필드

나머지 필드는 Text형식 필드

ROCKFISH_GENERAL_SELECT- 조회

ROCKFISH_GENERAL_INSERT- 입력

ROCKFISH_GENERAL_UPDATE- 수정

ROCKFISH_GENERAL_DELETE- 삭제

General Send 버튼 클릭 하여 테스트

Input Text값 위에서 아래로 순서대로

IDX,TEMP1,TEMP2,TEMP3 필드임

IDX는 정수 값을 입력

---------------------------------

Multipart - Multipart 테스트

Download - Download 테스트

Multipart,Download는 Service Type과 상관 없음

(참고) 로그인 처리는 되어 있어야 함

HTTP 전송테스트 프로그램으로 유효성 체크 미구현

공통 프로그레스바 등도 미구현

결과는 Alert or Toast 창 출력
